Arrays In javascript

The forEach() Method
forEach() method is anther way to loop over an array, or access each array element.
It Executes a callback function for each array element.

The callback function is invoked with 3 arguments.
1. The value of the element
2. The index of the element
3. The Array itself

The lastindexOf() method returns the index of the last occurence of an item in the array
