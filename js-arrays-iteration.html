Using forEach() Method

The forEach() method executes a provided function once for each array element,
It Have one paramater, a callback function

callback function accepts one to three arguments

  • value - the current element being processed
  • index - the index of the current element
  • array- the array itself

Using map() Method

The map() method creates a new array
each element in the new aaray is changed using the logic in the provided function.

provided function accepts one to three arguments

  • value - the current element being processed
  • index - the index of the current element
  • array- the array itself

Using filter() Method

The filter() method creates a new array.

The new array only contains the elements that passed the test implemented by the provided parameter function.

provided function accepts one to three arguments

  • value - the current element being processed
  • index - the index of the current element
  • array- the array itself
