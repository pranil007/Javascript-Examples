JavaScript Math Object Methods

-- What is an Math Object? --

The ceil() Method

The Math.ceil method rounds up a number to nearest whole number or integer.

The floor() Method

The Math.floor method rounds down a number to nearest whole number or integer.

The round() Method

The Math.roud method rounds a number to nearest whole number or integer.

The sqrt() Method

The Math.sqrt method rounds a square root of a number. It returns Nan when number is negative.

The cbrt() Method

The Math.sqrt method rounds a cube root of a number.

The abs() Method

The Math.abs method retruns a absolute value of number.

The max() Method

The Math.max method retruns a largest number from a given array elements.

The min() Method

The Math.min method retruns a smallest number from a given array elements.

The pow() Method

The Math.pow method retruns the base to the exponent power.
It takes 2 arguments: base, exponent.
